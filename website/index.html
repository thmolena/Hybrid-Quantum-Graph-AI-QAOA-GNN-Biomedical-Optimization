GNN-for-QAOA Demo

Notebooks (HTML exports)

The following exported notebook HTML files provide reproducible experiment artifacts, visual diagnostics, and summary results for the hybrid QAOA-GNN and biomedical workflows.

  • QAOA + GNN demo (MaxCut) — shows classical optimization vs GNN prediction, timing, and expectation landscapes.
  • Biomedical GNN demo (breast cancer) — shows GNN classification on real biomedical data, saved CSV, and evaluation metrics.
  • Combined demo - consolidated outputs and narrative connecting the optimization and biomedical components in a unified research pipeline.

Research Summary

This project demonstrates an integrated pipeline combining quantum algorithms (QAOA), graph neural networks (GNNs), and biomedical data analysis. Each notebook includes inputs, model design, outputs, performance indicators, visualizations, and reproducible code to support transparent evaluation of the proposed hybrid methodology.
