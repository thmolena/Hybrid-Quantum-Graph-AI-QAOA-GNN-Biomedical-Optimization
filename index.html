What the prediction means

The demo samples a graph G = (V, E), builds a simple node-feature view from the graph structure, and then asks the model for a good depth-1 QAOA parameter pair (γ, β). In practical terms, the GNN is learning a shortcut from graph structure to quantum-circuit settings, so it can propose angles without rerunning a full classical search loop for every new graph.

The value reported as Expected Cut is the average MaxCut score the QAOA state would produce under those angles. When the local API is running, the page uses the trained GNN through /predict; when it is not available, the page falls back to an in-browser depth-1 grid search so the demo still remains interactive.

MaxCut Objective
\[ C(z) = \sum_{(u,v) \in E} \frac{1 - z_u z_v}{2} \]
Depth-1 QAOA State
\[ |\gamma, \beta\rangle = U_B(\beta)\, U_C(\gamma)\, |+\rangle^{\otimes n} \]
Quantity Being Maximized
\[ \langle C \rangle = \langle \gamma, \beta | \, \hat{C} \, | \gamma, \beta \rangle \]

Research Summary

Approach

Graph-structured problems are solved by first learning structural representations with a Graph Convolutional Network (GCN), then using those learned embeddings to initialize QAOA circuit parameters, replacing expensive variational search with fast GNN inference.

Applications

  • MaxCut combinatorial optimization
  • Healthcare resource allocation
  • Biomedical experimental design
  • Network resilience analysis

Quick Start

pip install -r requirements.txt
python -m src.train \
